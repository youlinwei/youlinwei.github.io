读书|未来世界的幸存者

发表于 2019-06-16 | 更新于: 2019-06-16 | 分类于 读书笔记 | | 本文总阅读量 次

去年十月份的时候，我读过一遍阮一峰老师的《未来世界的幸存者》，当时读的比较快，没有细细品味。前两天，我又慢慢的品读了一遍，收获颇多。

这本书是阮一峰老师对于现实，职业，创业，未来的一些深入思考的整理总结，篇幅很短，但每一章读起来都是那么的耐人寻味。

作者认为：这个世界从短期来看是由政治主导的，从中期来看是由经济主导的，从长期来看则是由技术主导的。石器时代、铁器时代、工业革命、计算机革命。人类的每次划时代变革都是由技术主导的。

关于未来

回到这本书的书名，到底谁才是未来世界的幸存者？答案显而易见：掌握技术的资本家，他们才是这个世界真正的控制者。

未来世界大多数的职业都会被机器所替代，只有富有人格魅力和人性化的职业，才不会被机器所替代。大多数人都会被淘汰，社会阶层会进一步固化。

为了保持社会的稳定，娱乐性产业会得到空前的发展，比如游戏或者说是VR游戏。娱乐性产业的一个重要目的就是消耗掉人们由于无所事事而造成的精力过剩，这样就没有精力去犯罪了。

关于基因编辑

以色列历史学家尤瓦尔.赫拉利的著作《未来简史》和这本《未来世界的幸存者》中都提到了基因编辑这一可能会改变世界的技术。

随着基因编辑技术的发展和成熟，治愈性编辑和增强性编辑的界限将会模糊。未来会出现两种人类：一种人被基因编辑过，无论是身高，相貌，体能，智力都优于常人；另一种人则是普通大众。生理上的差异会进一步固化社会阶层，这种固化将是不可逆转的。

关于创业

创业的一个核心要点：不要在功能上竞争，因为这是争不过大公司的。

创造一个有竞争力的新产品，不要着眼于它的功能比别人多，而要着眼于它有一个截然不同的市场定位。

关于职业

大多数人只拥有职业，没有事业。雇佣制度天然就对工人不利，如果突然被辞了，工作没有了，接下来该怎么办？作者认为：我们应该制定一个B计划，计划好万一工作没有了，接下该干什么，并不断为之准备着。

提前指定自己的退休计划，越早越好。职业生涯的开始，就应该制定好自己的退休计划，并为之准备着。

不要奢望国家的养老基金，未来随着老龄化的加重，老年人越来越多，养老金的支出远远大于收入，会产生一个巨大的养老金赤字。再加上医疗技术的发展，人们越来越长寿。对于老年人来说，最尴尬的事情莫过于，钱花完了，人还没死。

通货膨涨一直都存在，每个人都应该掌握一点投资理念，哪怕只是为了保值。
