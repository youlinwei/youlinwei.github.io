读书|深入了解我们的潜意识

发表于 2019-06-19 | 更新于: 2019-06-19 | 分类于 读书笔记 | | 本文总阅读量 次

最近两天在读一本书：《思考致富》。这本书首版出于1937年，销量超过1500万册，是一本十分著名的励志书籍。

为什么读这本书呢？因为一个欲望，一个致富的欲望。

作者开篇即提出：一切成就，一切辛劳所得的财富，最初都不过源于一种想法。当我看到这句话时就被震住了，其实自己的每一个行为，最初何尝不只是源于一个想法。

作者主要从欲望、信念、自我暗示、专业知识、想象力、精心策划、决心、毅力、性转换、智囊团、潜意识思维、大脑、第六感这十三个方面分析致富的方法论。

我主要想聊的是：潜意思

意识和潜意识

人的思维主要分为意识和潜意思，当我们清醒冷静的时候，意识掌控着大脑。意识只会在我们有意而为之的时候控制大脑，其他时候皆是由潜意识控制，因为潜意识一天24小时都是在运作的。

之所以称为潜意识，就是因为这是无法控制的。而潜意识对一个人的影响是无比巨大的，一想到自己被一个称为“潜意识”的家伙深刻影响着，而自己却控制不了它，这本身就是一件很可怕的事情。

潜意识的运作

潜意识就像一片肥沃的土地，你种什么，它就长什么；你什么也不种，它依然会长，只不过会长的像杂草丛一般。

种向潜意识这块土地的，正是我们每天无时无刻不存在的想法。很多很多的想法，都在种植着潜意识这块土地。如果某一种想法特别的强烈，那它在潜意识里种下的种子也就越多，种子渐渐长大成树，潜意识受到这些树的影响，就会反过来作用于人的行为，从而改变人的行为。

如何影响潜意识

既然控制不了潜意识，那如何影响潜意识呢？至少不要让潜意识和自己作对，最好是能够帮助到自己。

影响潜意识只有一个办法：自我暗示。

人主要有两种情绪：消极和积极。消极情绪会自动流入潜意识之中，并且能够顺畅地进入潜意识思维。而积极情绪则必须通过自我暗示，被动地注入到潜意识这块土地上。

积极的情绪主要有以下7种：欲望、信念、爱、性、激情、浪漫和希望。同样的，消极情绪也主要有7种：恐惧、嫉妒、憎恨、报复、贪婪、迷信和愤怒。

影响潜意识的关键在于：养成采纳并运用积极情绪的习惯，并通过自我暗示，让这些积极情绪进入自己的潜意识思维，通过这种方法，不断影响自己的潜意识，让潜意识成为自己得力助手。

读书的另一种意义

每一个成功的人，没有一个是不爱读书的，但读书真的能成功吗？答案肯定是不一定。

书那么多，读了一本又一本，仿佛一点用都没有，真的是一点用都没有吗？

通过理解潜意思的作用，我突然发现，读书的核心之处在于：读书这个动作本身是积极的，它能产生积极的情绪：欲望、信念和希望。反复进行读书这个动作能够源源不断地产生这种积极情绪，并通过自我暗示注入潜意识，从而不断影响潜意识。

这才是读书的另一种意义。
